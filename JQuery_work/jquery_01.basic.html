제이쿼리는 자바스크립트 라이브러리

제이쿼리 적용하는 방식 : CDN방식 / 다운로드 방식 => 외부의 js파일을 불러와서 실행하는 js라이브러리 -> jquery는 함수형으로 모든 것이 구현되어 있음. DOM을 관리하는데 편한 함수들이 많다

jquery적용하는 방식 : $(예약어,변수)를 사용해서 처리함. 주로 DOM(element)를 불러와서 처리하는 곳에 자주 사용

사용법 : $("선택자(css선택자와 매우비슷)").함수명(매개변수,....)

제이쿼리이용

$("선택자") DOM(element)를 가져와서 처리함.
