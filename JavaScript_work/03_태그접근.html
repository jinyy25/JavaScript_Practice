html페이지안에 있는 태그에 접근하기

자바스크립트에서 html태그는 모두 .... 객체화되어있음(DOM)
그 객체들을 관리는 document객체에서 함
각 태그(태그객체)에 접근하려면 document가 제공하는 함수(메소드)를 이용
1. id속성을 이용해서 가져오는 함수(메소드)
- document.getElementById("아이디속성값");
- 한개의 태그만 가져옴 * id값이 두개 이상이면 무조건 맨 위태그 한개만 가져옴
2. class속성을 이용해서 가져오는 함수(메소드)
- document.getElementByClassName("클래스속성값");
- 다중값이 있어 태그들을 배열형식으로 가져옴
3. name속성을 이용해서 가져오는 함수(메소드)
- document.getElementsByName("name속성값");
- 다중값이 있어 태그들을 배열형식으로 가져옴
4. Tag 이름을 이용해서 가져오는 함수(메소드)
- document.getElementsByTagName("태그명");
- 다중값이 있어 태그들을 배열형식으로 가져옴
