데이터출력하기

데이터(value)를 출력하는 방법은 여러가지가 있음
1. 페이지내부에 출력하는 방법
document객체를 이용한 출력
1) document.write("데이터");
2) tag객체.innerHTML="데이터";

2. 브라우저에 출력하는 방법(팝업, console창)
window객체가 이용하는 함수를 이용
1)window.alter("데이터") ->팝업창
2)window.console.log("데이터") -> 브라우저 console창

데이터입력받기

페이지이용자에게 데이터입력 받기
1. window객체가 제공하는 함수 이용하는
1) confirm() : 진위형으로 맞는지 아닌지를 확인할때 사용하는 입력방법 -> true/false값을 받음
2) prompt() : 클라이언트에게 작성하는 창을 띄워줌 -> 작성한 내용을 받음
2. 태그를 통해 입력받는 방법
1) input태그를 이용해서 받은 값을 이용하는 것
